# 他妈的我们吃什么！

一个帮你随机决定今天吃什么的小工具。

## 环境要求

- Go 1.22 或更高版本

## 运行

```bash
go run .
```

服务器启动后，打开浏览器访问 [http://localhost:8080](http://localhost:8080)。

## 构建并运行

```bash
go build -o wtfeat .
./wtfeat
```

## 运行测试

```bash
go test ./...
```

## 功能说明

- 点击"开始"按钮随机选择一家餐厅
- 每次会话最多可点击 3 次
- 超过 3 次后会提示"吃个锤子吃，你狗东西就是不饿"
